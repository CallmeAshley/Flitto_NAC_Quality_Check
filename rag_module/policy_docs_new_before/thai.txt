[Locale: th]
1. 숫자 표기 (Numeric Format)

Decimal separator: '.' (dot)
Grouping separator: ',' (comma)
Example: 1,234.56
설명: 영미식 숫자 표기법을 사용합니다. 태국어 고유 숫자 시스템(thai)도 지원하지만 기본은 라틴 숫자(latn)입니다.

2. 화폐 표기 (Currency Format)

Symbol format:
• Pattern: "¤#,##0.00"
• 예시: "฿1,234.56"
• 설명: 통화 기호(¤)가 숫자 앞에 공백 없이 표기됩니다.
ISO Code format:
• Pattern: "{0} ¤¤"
• 예시: "1,234.56 THB"
• 설명: 통화 코드는 숫자 뒤에 공백과 함께 표기됩니다.

3. 날짜 및 요일 표기 (Date Format)

Full: "วันจันทร์ที่ 28 กรกฎาคม 2025"
Long: "28 กรกฎาคม 2025"
Medium: "28 ก.ค. 2025"
Short: "28/7/25"
날짜 순서 (Order): Day / Month / Year
요일 위치: Full 포맷에서는 날짜 앞에 "วัน + 요일 + ที่" 패턴 사용 (예: วันจันทร์ที่)

4. 시간 표기 (Time Format)

Full: "14 นาฬิกา 30 นาที 45 วินาที เวลาในประเทศไทย"
Long: "14 นาฬิกา 30 นาที 45 วินาที GMT+7"
Medium: "14:30:45"
Short: "14:30"
설명: 24시간제를 사용하며, Full/Long 포맷에서는 태국어 시간 단위(นาฬิกา=시간, นาที=분, วินาที=초)를 사용합니다. 날짜-시간 연결에는 "เวลา"(시간)을 사용합니다.
