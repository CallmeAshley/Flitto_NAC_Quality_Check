1. 숫자 표기 (Numeric Format)

Decimal separator: '.' (dot)
Grouping separator: ',' (comma)
Example: 1,234.56
설명: 기본적으로는 영미식 숫자 표기를 사용하지만, 중국어 특유의 큰 수 단위체계를 함께 사용합니다.
큰 수 단위: 萬(만=10,000), 億(억=100,000,000), 兆(조=1,000,000,000,000)

2. 화폐 표기 (Currency Format)
Symbol format

Pattern: "¤#,##0.00"
예시: "NT$1,234.56" 또는 "$1,234.56"
설명: 통화 기호(¤)가 숫자 앞에 공백 없이 표기됩니다.

ISO Code format

Pattern: "{0} ¤¤"
예시: "1,234.56 TWD"
설명: 통화 코드는 숫자 뒤에 공백과 함께 표기됩니다.

3. 날짜 및 요일 표기 (Date Format)

Full: "2025年7月28日 星期一"
Long: "2025年7月28日"
Medium: "2025年7月28日"
Short: "2025/7/28" 또는 "2025-7-28"
날짜 순서 (Order): Year年Month月Day日 (중국식)
요일 위치: Full 포맷에서는 날짜 뒤에 요일을 명시하며 공백으로 구분 (예: 2025年7月28日 星期一)
대체 형식: yyyy-mm-dd, yyyy/mm/dd, mm/dd 형식도 사용 가능

4. 시간 표기 (Time Format)
기본 형식

Full: "上午2:30:45 [中華民國標準時間]"
Long: "上午2:30:45 [CST]"
Medium: "上午2:30:45"
Short: "上午2:30"
설명: 12시간제와 24시간제 모두 사용 가능하며, 날짜-시간 연결에는 공백을 사용합니다.

대만식 시간대 구분 (CLDR 기준)

凌晨 (링천): 0:00-5:00 (새벽)
早上 (자오상): 5:00-8:00 (이른 아침)
上午 (상우): 8:00-12:00 (오전)
中午 (중우): 12:00-13:00 (정오)
下午 (하우): 13:00-19:00 (오후)
晚上 (완상): 19:00-24:00 (저녁/밤)
