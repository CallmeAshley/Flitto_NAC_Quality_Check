[Locale: ja]

1. 숫자 표기 (Numeric Format)
- Decimal separator: '.' (dot)
- Grouping separator: ',' (comma)
- Example: 1,234.56
- 설명: 소수점은 점(.), 천 단위 구분은 쉼표(,)를 사용합니다.

2. 화폐 표기 (Currency Format)

- Symbol format:
  • Pattern: "¤#,##0.00"
  • 예시: "￥1,234.56"
  • 설명: 통화 기호(￥)는 숫자 앞에 붙으며 공백 없이 붙습니다.

- ISO Code format:
  • Pattern: "¤¤ #,##0.00"
  • 예시: "JPY 1,234.56"
  • 설명: 통화 코드(JPY)는 숫자 앞에 공백(U+00A0)과 함께 표기됩니다.

3. 날짜 및 요일 표기 (Date Format)

- Full: "2025年7月28日月曜日"
- Long: "2025年7月28日"
- Medium: "2025/07/28"
- Short: "25/07/28"

- 날짜 순서 (Order): **Year / Month / Day**
- 요일 위치: Full 포맷에서는 날짜 뒤에 요일을 명시함

4. 시간 표기 (Time Format)

- Full: "HH時mm分ss秒 日本標準時"
- Medium: "HH:mm:ss"
- Short: "HH:mm"

- 설명: 24시간제를 기본으로 사용하며, Full 포맷에서는 타임존(예: 日本標準時)을 포함합니다.
