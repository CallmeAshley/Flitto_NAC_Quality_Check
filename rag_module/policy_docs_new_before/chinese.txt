[Locale: zh]
1. 숫자 표기 (Numeric Format)

Decimal separator: '.' (dot)
Grouping separator: ',' (comma)
Example: 1,234.56
설명: 소수점은 점(.)을, 천 단위 구분은 쉼표(,)를 사용합니다. 서구식 숫자 표기법을 따릅니다.

2. 화폐 표기 (Currency Format)

Symbol format:
• Pattern: "¤#,##0.00"
• 예시: "¥1,234.56"
• 설명: 통화 기호(¤)는 숫자 앞에 바로 붙여서 표기됩니다.
ISO Code format:
• Pattern: "{0} ¤¤"
• 예시: "1,234.56 CNY"
• 설명: 통화 코드는 숫자 뒤에 공백과 함께 표기됩니다.

3. 날짜 및 요일 표기 (Date Format)

Full: "2025年7月28日星期一"
Long: "2025年7月28日"
Medium: "2025年7月28日"
Short: "2025/7/28"
날짜 순서 (Order): Year / Month / Day
요일 위치: Full 포맷에서는 날짜 뒤에 요일을 명시함 (예: 星期一)

4. 시간 표기 (Time Format)

Full: "中国标准时间 14:30:45"
Long: "CST 14:30:45"
Medium: "14:30:45"
Short: "14:30"
설명: 24시간제를 기본으로 사용하며, Full 포맷에서는 타임존(예: 中国标准时间)을 포함합니다.
